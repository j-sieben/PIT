print_call_stack 

Procedure print_call_stack(p_call_stack in call_stack_type,
                           p_template   in varchar2)

Helper to print the content of a call stack

Usage
Templates are defined as parameters and referenced here to control the formatting of the call stack. After formatting, procedure PRINT is called to print the call stack.

Parameters
p_call_stack   Instance of call stack to print
p_template   Name of the template to be used to format the output.
