Table PIT_TRANSLATABLE_ITEM

Central translatable item repository in various languages.
Columns
Name Type Optional Default Comments
PTI_ID VARCHAR2(128)     ID of the translatable item, part of the PK in conjunction with PIT_PML_NAME
PTI_PML_NAME VARCHAR2(128)     Part of the PK, reference to PIT_MESSAGE_LANGUAGE. Taken from Oracle v$nls_valid_values
PTI_PMG_NAME VARCHAR2(128)     Part of the PK, reference to PIT_MESSAGE_GROUP, criteria to group messages and translatables items per group
PTI_NAME VARCHAR2(800) Y   Name. Optionally contains replacement anchors
PTI_DISPLAY_NAME VARCHAR2(800) Y   Display name. Used for UI display purposes. Optionally contains replacement anchors
PTI_DESCRIPTION CLOB Y   Help text or further description. No replacement anchors allowed
PTI_UID VARCHAR2(128) Y CASE "PTI_PML_NAME" WHEN 'GERMAN' THEN "PTI_ID" ELSE NULL END Virtual column as a surrogate primary key for references
PTI_UPMG VARCHAR2(128) Y CASE "PTI_PML_NAME" WHEN 'GERMAN' THEN "PTI_PMG_NAME" ELSE NULL END Virtual column as a surrogate primary key for references
