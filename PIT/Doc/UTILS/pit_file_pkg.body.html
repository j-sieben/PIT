write_call_stack 

Procedure write_call_stack(p_call_stack in call_stack_type,
                           p_template   in varchar2)

Helper to format call stack entries to write them to the log file

Usage
Templates are defined as parameters and referenced here to control the formatting of the call stack. After formatting, procedure WRITE_TO_FILE is called to append the call stack to the log file.

Parameters
p_call_stack   Instance of call stack to be printed
p_template   Name of the template to be used to format the output.
