Table PIT_TABLE_CALL_STACK

table for trace and time information
Columns
Name Type Optional Default Comments
CLS_ID NUMBER     Primary key, deliverd with the log message.
CLS_LOG_DATE TIMESTAMP(6) Y   Time at which the entry was created. Required for PIT_TABLE.purge method
CLS_CALL_LEVEL NUMBER Y   Increment for each step of the call stack
CLS_ACTION VARCHAR2(5) Y   Activity of the current step: method name, line-number or user defined action
CLS_SESSION_ID VARCHAR2(256) Y   Session-Id, required to attach log messages to sessions
CLS_USER_NAME VARCHAR2(128) Y   Connected database user
CLS_MODULE_NAME VARCHAR2(128) Y   Package name
CLS_METHOD_NAME VARCHAR2(128) Y   Method name of the package
CLS_ELAPSED NUMBER Y   Time consumed by the procedure (w/o time spent in called procedures)
CLS_ELAPSED_CPU NUMBER Y   CPU-time consumed by the procedure (w/o time spent in called procedures)
CLS_TOTAL_TIME INTERVAL DAY(2) TO SECOND(6) Y   Time consumed by the procedure (including time spent in called procedures)
CLS_TOTAL_TIME_CPU NUMBER Y   CPU-time consumed by the procedure (including time spent in called procedures)
