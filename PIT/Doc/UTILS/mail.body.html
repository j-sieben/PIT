mail_server_accessible 

Function mail_server_accessible return boolean

Method to check whether an ACL entry for this mail server is accessible

Returns
Flag that indicates whether the mail server was successfully touched (TRUE) or not (FALSE). If a UTL_TCP.NETWORK_ERROR is received, the return value indicates whether the exception relates to a LISTENER. If so, TRUE is returned and FALSE in other cases.

initialize 

Procedure initialize

Initialization method.

Usage
Initializes global variables and tries to detect whether the package is in a usable state. This requires that
-   the mail server is accessible via ACL
-   the mail server is accessible and responds to a touch

Function string_to_address_tab(p_address_list in varchar2)

Is called from the simplified SEND_MAIL method to allow for more than one recpient without having to create an instance of ADDRESS_TAB externally. Instead, a semicolon-delimited list of addressees is sufficient

Parameters

Returns
Instance of ADRESS_TAB containing the addresses
