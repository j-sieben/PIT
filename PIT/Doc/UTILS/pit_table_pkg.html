purge 

Procedure purge(p_date_until             in date,
                p_severity_greater_equal in number default null)

Method to purge log information from tables PIT_TABLE_LOG, PIT_TABLE_CALL_STACK and PIT_TABLE_PARAMS

Usage
Method implements the PURGE member procedure of type PIT_TABLE and removes entries from the tables according to the filter set by P_DATE_UNTIL and P_SEVERITY_GREATER_EQUAL. Entries from PIT_TABLE_CALL_STACK and PIT_TABLE_PARAMS are not filterd by P_SEVERITY_GREATER_EQUAL but only by date.

Parameters
p_date_until   Date up to which the log entries are to be deleted
[p_severity_greater_equal]   Severity up to which the log entries are to be deleted
