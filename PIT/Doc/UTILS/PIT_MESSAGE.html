Name Type Optional Default Comments

PMS_DESCRIPTION CLOB Y

PMS_CUSTOM_ERROR NUMBER Y   Optional pointer to an oracle error number. If null, an exception will be generated. To define custom errors, use null.
PMS_ACTIVE_ERROR NUMBER Y   If a custom error number of -20000 is used, this field contains the actual error number as processed by PIT_ADMIN.CREATE_MESSAGE_PACKAGE.
