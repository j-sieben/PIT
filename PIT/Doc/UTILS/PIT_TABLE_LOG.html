Name Type Optional Default Comments

LOG_SESSION_ID VARCHAR2(256) Y   Session-Id, required to attach log messages to sessions
LOG_USER_NAME VARCHAR2(128) Y   Connected database user
LOG_MESSAGE_TEXT CLOB Y   Log message text
LOG_MESSAGE_NAME VARCHAR2(128) Y   Name of the PIT message. Can be used together wit MESSAGE_ARGS to construct a log in any language
LOG_MESSAGE_ARGS MSG_ARGS_CHAR Y   MSG_ARG instance. Can be used together wit MESSAGE_NAME to construct a log in any language
LOG_MESSAGE_STACK VARCHAR2(4000) Y   Error stack in case of error
LOG_MESSAGE_BACKTRACE VARCHAR2(4000) Y   Call stack in case of error
LOG_SEVERITY NUMBER Y   Severity of the message logged
