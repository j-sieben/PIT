Procedure authenticate(p_conn        in out nocopy utl_SMTP.connection,
                       p_hash_method in varchar2,
                       p_user_name   in varchar2,
                       p_password    in varchar2)

Method to authenticate with a CRAM SMTP server like Exchange

Usage
Procedure will authenticate p_conn with the credentials provided and return the connection

Parameters
p_conn   Parameter for the actual SMTP-connection
p_hash_method   One of the package constants to identify the authentication method
p_user_name   Name of the user to authenticate
p_password   Authentication users password
